berlin-opengeo
==============

An Open Source geocoder for Berlin

# Installation
    git clone git@github.com:g-div/berlin-opengeo.git
    cd berlin-opengeo
    ./setup.sh
    npm install

# Start
    npm start
    
or

    node api/api.js

### Requirements:
    wget
    7zip
    MongoDB
    Node.js
    NPM
