berlin-opengeo
==============

An Open Source geocoder for Berlin.

## Installation
    git clone git@github.com:g-div/berlin-opengeo.git
    cd berlin-opengeo
    chmod +x setup.sh
    ./setup.sh
    npm install

## Start
    npm start
    
or
    
    node api/api.js

then open your browser at [http://localhost:9987/docs/](http://localhost:9987/docs/) to visit the API documentation


### Requirements:
- wget
- 7zip
- MongoDB
- Node.js
- npm
