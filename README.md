berlin-opengeo
==============

An Open Source geocoder for Berlin

## Installation
    git clone git@github.com:g-div/berlin-opengeo.git
    cd berlin-opengeo
    ./setup.sh
    npm install

## Start
    npm start
    
or

    node api/api.js

then visit http://localhost:9987/docs/ to get the API documentation


### Requirements:
    wget
    7zip
    MongoDB
    Node.js
    NPM
